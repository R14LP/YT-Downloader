# Privacy Policy for Grabber Helper

**Effective Date:** April 15, 2026

## 1. Information Collection and Use
The Grabber Helper extension does not collect, store, or transmit any personal data, user information, or browsing history to any external servers. 

## 2. Permissions
The extension requires certain permissions (such as `activeTab` and `nativeMessaging`) strictly to capture the media URL of the currently active tab and securely transmit it to the user's locally installed Grabber desktop application.

## 3. Contact
If you have any questions regarding this privacy policy, please open an issue on our GitHub repository.
